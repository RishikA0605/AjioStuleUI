🛍️ E-Commerce Frontend Project

This is a modern and responsive e-commerce frontend web application inspired by AJIO. It showcases product listings with features like search, filtering, and category navigation.

🚀 Features:
- Product listing with dynamic data
- Search functionality
- Category-based navigation (Men, Women, Kids, etc.)
- Top-rated products section
- Responsive design for all devices
- Clean and modern UI

🛠️ Tech Stack:
- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- REST API (FakeStore API)

📌 Purpose:
This project was built to practice frontend development skills including React components, state management, API integration, and UI design.
